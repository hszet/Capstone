# E-commerce Customer Segmentation & Product Affinities (Olist)

Capstone project for General Assembly's Data Analytics course.

## Overview

This project analyses the Brazilian Olist e-commerce dataset to:
- Segment customers based on recency, frequency, monetary and behavioural features
- Identify high-value customer groups
- Uncover product affinities to inform cross-sell and bundling strategies

## Tech stack

- Python (pandas, numpy, scikit-learn)
- SQL
- Jupyter Notebooks
- Tableau / Power BI for dashboards

## Key methods

- Feature engineering (RFM-style metrics, delivery and order behaviour)
- Clustering (K-Means, GMM, HDBSCAN) with silhouette / inertia comparison
- Market-basket analysis (support, confidence, lift)

## Outputs

- Customer segment definitions and profiles
- Product bundle recommendations
- Dashboards for stakeholders to explore segments and performance

## Files

- `notebooks/`: analysis notebooks
- `data/`: sample or reference to source data
- `report/Capstone_Technical_Report.pdf`: full technical write-up
- `dashboards/`: screenshots or links to Tableau / Power BI
